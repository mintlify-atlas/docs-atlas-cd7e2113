## Using Templates

### List Available Templates

```bash
# Human-readable list
open-agreements list

# Machine-readable JSON (for AI agents)
open-agreements list --json
```

### Fill a Template

```bash
# Using a JSON data file
open-agreements fill common-paper-mutual-nda -d values.json -o nda.docx

# Using inline --set flags
open-agreements fill common-paper-mutual-nda \
  --set party_1_name="Acme Corp" \
  --set party_2_name="Beta Inc" \
  --set effective_date="2026-03-15"
```

### Validate a Template

```bash
# Validate one template
open-agreements validate common-paper-mutual-nda

# Validate all templates
open-agreements validate
```
